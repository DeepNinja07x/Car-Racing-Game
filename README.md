# Car-Racing-Game

The Car-Racing-Game is a web-based program which resonates to ols school _Car Game_. The User needs to control the moving __Car__ with the _Arrow Keys_ to dodge the __Obstacles__
(_Blue Boxes_) in order to score more _Points_. Following are the controls list for this Game:
- UP Arrow : Forward
- DOWN Arrow : Backward
- RIGHT Arrow : Move Sideways(Right)
- LEFT Arrow : Move Sideways (Left)
